前言：

  用Hexo框架搭建博客并没多复杂，简单总结也就只有下载工具、安装Hexo、安装主题并配置，仅此而已。当然，想要弄成自己想要的样式，那是属于魔改的操作了。如果只是用于记录一些学习心得之类的，找一个喜欢的主题就完全可以了。本文章很基础，也算是个人备忘，话不多说，就开始吧。

准备工作：

  • 操作系统：Windows11
  • Node
  • Git
  • Hexo
  • idea（推荐使用 Visual Studio Code）
  • 一个 GitHub 帐号

一些命令使用

hexo cl：清除本地缓存；
hexo g：生成静态文件；
hexo s: 运行；
hexo d：一键部署(后面会说)。
