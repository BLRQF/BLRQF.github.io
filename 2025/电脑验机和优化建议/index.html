电脑验机和优化建议

发表于2025-02-28|更新于2025-08-25|教程电脑教程
|总字数:1.6k|阅读时长:4分钟|浏览量:

验机有什么用

  如果买新电脑前有自己去在网上找攻略，那一定会看到很多博主说第一步是 “验机”。简单来，说其目的就是为了看自己有没有被坑，是否是原装，配置是否被更换等等。主要包括检查外观、开机、跳过联网、查看配置、基础设置等等。其中 “跳过联网” 是非常关键的一步，因为电脑联网会自动激活系统，一但激活，出现问题后一般是不给退的。即便是后面重装系统啥的，跳过联网可以省去一部分相当繁琐的步骤（本人也是头铁的尝试过不跳过，可谓是相当后悔）。

检查外观

  虽然我觉得这个算是废话，但还是仔细说说吧：

  1. 首先观察快递包装是否破损，然后再打开内包装。
  2. 内包装需要特别注意一下，一般笔记本的封条都经过特殊设计，上面会有品牌logo，撕下后会留下印迹，新机的是规则的，同时也要注意封条是否平整、有气泡等等。
  3. 再看看产品信息标签，是不是自己所买的型号，最后取出电脑，最好是可以带个手套，防止商家甩锅。观察外观是否有划痕，看看螺丝有没有拧过的痕迹。值得注意的是，有的电脑你买的时候可能是加装了内存的，如果是那种，那么肯定是拆过的（并不会影响啥，但还是不太建议买那种，自己买固态加上去更有性价比点，也能避免被配置坑的问题）

打开电脑

开机：

  1. 不插电，按一下电源键，如果没开机，那说明电脑没问题，是第一次使用。因为笔记本电脑厂商有一个运输模式，所以第一次使用只有在插上电后才会开机。（也有少数没有运输模式，如惠普，机械革命）
  2. 通电，开机，进入系统启动界面，前面按自己需要设置，一直到让自己联网这一步，不要联网！ 顺便提一下，不论是轻薄本还是游戏本，都是在插电情况下才会发挥最大性能，如果没有外出，请插电使用！ 插电使用不会减少电脑寿命，但是不插电会。我是真遇到有人不插电使用，然后来问为啥很卡的。

跳过联网：

  1. 按住键盘上的Shift+F10或者Shift+Fn+F10，此时会弹出cmd窗口，在其中输入 然后按下回车键会自动重启，等待就好。（如果输入不了，请用鼠标点一下cmd窗口）。
  2. 重启后联网界面就会有一个 “我没有internet链接” 选项，然后选择继续执行受限设置就跳过了。进入系统后，不要联网！

验机

下载工具：

  • 将图吧工具箱下载到U盘或者手机上，进入系统后将它传入电脑中并打开。（手机使用数据线与电脑连接）。打开如下：

查看硬件信息：

  1. 在左边选择硬件信息即可看自己配置，看看是否有出入就行。
  2. 再选择磁盘工具，双击 “Disklnfo(64位)” 打开，看一下通电次数和健康度，由于出厂会有调试，通电次数在个位数健康度为100% 就代表是全新的。
  3. 还是在磁盘工具，双击 “DiskMark(64位)” 打开，在设置里改为 “NVMe SSD”，如下图：

    然后将上方依次改为 1，1GiB，C盘。再点击 “All” 进行测试，最低要求：
    REND4K Q1T1：读 ≥ 50MB/s；写 ≥ 100MB/s 。
    SEQ1MQT1 和 SEQ1MQ8T1：标称值的 80% 。
  4. 点击左侧CPU工具，选择 “AIDA64” 打开，在电源管理中查看 “损耗程度”，0% 和 1% 为正常。

检测屏幕：

  • 点击左侧屏幕工具，双击打开 “坏点与漏光测试”，然后观察有没有屏幕坏点，点击鼠标切换画面，按ESC键退出。

查看键盘：

  • 在左侧外设工具中双击打开 “Keyboard Test Utillity”，然后按下键盘按键，没坏的就行。

烤机：

  • 其实按照上面检测出来就不会有太大问题了，加上我个人并不烤机，所以这里只提一下，想要弄的去B站搜一下，我就不多赘述了。

激活：

  • 联网自动激活（顺便提一下，用户名最好是英文，中文的话有时候会出问题，如蓝屏，死机，闪退，进不去程序啥的）。

一些基本设置

  • 设置 → 电源和电池 → 电源模式：选择 最佳性能 。
  • 设置 → 蓝牙和其他设备 → 鼠标 → 其他鼠标设置 → 指针选项 → 提高指针精准度：关 。
  • 然后就是删除一些不用的软件，改 存储感知 那些，在之前文章Windows11优化汇总中有。
