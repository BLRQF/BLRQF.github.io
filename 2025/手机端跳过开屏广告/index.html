开屏广告

  即打开软件时的广告，当下已经是非常普遍了，像什么“点击跳过”，“摇一摇切换软件”，甚至很多人调侃“打开软件根本不敢动”，“干脆改成眼睛在看屏幕就跳转呗”。在之前，有一款名为李跳跳的软件可以用来跳过，软件就只有一个开关，可以说是非常简洁，对用户极为友好了。但由于众所周知的原因，作者也不再更新了。以下分享个人在使用的方法，软件是开源的，不用担心安全问题

GDK

点击→GDK官网首页可以看一下作者介绍，和软件详情。
看到如下小黑子就是：

使用步骤

  1. 点击→下载地址选择正式版即可
  2. 打开后进入 GKD 的首页, 要想 GKD 正常工作需要授权无障碍权限，点击无障碍权限右侧的 授权 按钮即可前往设置的无障碍界面授权，以 Xiaomi HyperOS 为例
  3. 订阅
    进入软件，点击下方 订阅 → 右下角 “＋”，添加规则链接

规则

  作者并不提供任何规则，需自行添加本地规则，或者通过订阅链接的方式获取规则，这边就使用订阅链接方式。
规则列表：gkd-subscription，由于位于 Github 会出现打不开的情况，下方列出几个

  1. AIsouler/GKD_subscription：
    https://registry.npmmirror.com/@aisouler/gkd_subscription/latest/files/dist/AIsouler_gkd.json5
  2. 由 Adpro-Team 维护的 GKD 订阅规则：
    https://registry.npmmirror.com/@adpro/gkd_subscription/latest/files/dist/Adpro_gkd.json5
  3. 由 aoguai 维护的 GKD 第三方订阅规则
    1. GitHub 源（推荐）:
      https://raw.githubusercontent.com/aoguai/subscription/custom/dist/aoguai_gkd.json5
    2. npmmirror 源：
      https://registry.npmmirror.com/@aoguai/subscription/latest/files/dist/aoguai_gkd.json5
    3. gitmirror 源：
      https://raw.gitmirror.com/aoguai/subscription/custom/dist/aoguai_gkd.json5
    4. jsDelivr 源：
      https://cdn.jsdelivr.net/gh/aoguai/subscription@custom/dist/aoguai_gkd.json5

注意事项

  1. 配置完成后可在应用下方点击 应用 ，对相关应用设置除开屏广告外的功能。
  2. 规则类别里可以批量开启某一类规则，建议按需开启规则，不要无脑开启规则，开启过多规则可能造成规则阻塞从而导致规则触发缓慢甚至不触发，并且规则开启越多耗电也越多
